# 第一卷《青礁庶子》卷纲

> 目标字数：约 30 万字。  
> 建议章节数：40 章。  
> 核心任务：立刘枫、立青礁岛、立无名古树、立《青木生息诀》、立许潮、立刘岳、立海间宗外门考核。

---

## 卷一总目标

- **人物目标**：让读者认识刘枫——沉默、隐忍、温良，但不是任人宰割。
- **世界目标**：立住东海·万岛海·青礁岛的生活质感。
- **金手指目标**：完成“长年梦境 → 濒死被引 → 树下安睡 → 觉醒空间 → 第一颗果”的完整链条。
- **关系目标**：让读者喜欢许潮，记住刘清意，讨厌刘岳。
- **剧情目标**：刘枫通过海间宗外门考核，离开刘家，但不彻底决裂。
- **道心目标**：刘枫第一次明白：光会忍，不会换来自由。

---

## 三幕结构

| 幕 | 章节 | 核心内容 | 结尾状态 |
| -- | ---- | -------- | -------- |
| 第一幕 | 第 1-10 章 | 青礁岛、刘家、梦中古树、许潮与刘清意 | 刘岳设局，刘枫被迫去近海险地 |
| 第二幕 | 第 11-20 章 | 近海遇险、濒死、被引入空间、获得《青木生息诀》 | 刘枫隐瞒伤愈，开始秘密修炼 |

## 卷一关键伏笔清单

- **无名古树伏笔**：长年梦境、树下凹坑、衣襟陌生泥土、树叶轻颤、叹息、“醒了”。
- **功法伏笔**：《青木生息诀》不是自悟，是被引导；木灵力能稳定阵纹。
- **人物伏笔**：许潮必须讨喜；刘清意不能像老爷爷；刘岳不拖成长线。
- **地理伏笔**：青礁岛、万岛海、海间宗、东海潮汐规则。
- **未来伏笔**：旧木簪可作为母亲线唯一实物；旧阵符可引出刘枫阵法路线。

## 写作禁忌

- 不要让刘枫第一卷就大杀四方。
- 不要让刘岳成为长期宿敌，他只是第一块绊脚石。
- 不要解释无名古树的来历。
- 不要让空间里出现人形、声音、师尊。
- 不要让许潮过早卷入修仙主线，他主要是童年与人性锚。
- 不要让外门考核写成纯比赛，要写出海岛、潮汐、妖兽、阵法的地域特色。
