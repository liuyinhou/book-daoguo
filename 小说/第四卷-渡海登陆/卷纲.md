# 第四卷《渡海登陆》卷纲

> 目标字数：约 35 万字。  
> 建议章节数：44 章。  
> 核心任务：刘枫第一次离开东海、踏入神州；正式见到神州大宗气象；宋清河与苏凌登场；刘枫结丹成功但继续隐藏。

---

## 卷四总目标

- **地图目标**：从东海过渡到神州东岸大港——**观海城**。
- **身份目标**：刘枫从“海间宗弟子”变成“外来修士”，第一次承受神州体系的身份压制。
- **人物目标**：宋清河正式登场，苏凌以化名“苏影”伪装登场，三角格局初成。
- **试炼目标**：参加神州沿海联合试炼——**潮升试**。
- **遗迹目标**：探索大玄天朝旧海灯，第一次将东海航路、神州秩序、大玄伏笔连接起来。
- **修行目标**：刘枫以试炼灵物 + 阵悟 + 果实代价三者合一，结成金丹。

---

## 核心命题

> 刘枫终于踏上神州，却发现所谓“大地方”，只是把门槛修得更高。

---

| 幕 | 章节 | 核心内容 | 结尾状态 |
| -- | ---- | -------- | -------- |

## 主要新增人物

| 人物 | 定位 | 作用 | 后续 |
| ---- | ---- | ---- | ---- |
| **宋清河** | 神州大宗观察者/护道人 | 秩序侧镜像；在观海城危机中救人登场 | 长期参照物，不作伪君子 |

## 卷四关键伏笔清单

- **宋清河伏笔**：他真守规矩，不是伪君子；刘枫认可但不依附。
- **苏凌伏笔**：化名苏影同行，魔道手段一闪即收；结尾暗示“苏凌入境”。
- **大玄伏笔**：旧海灯残音、“大玄海门，照夜镇潮”、镇海盟约。
- **无名古树伏笔**：结丹服果时古树异动，青木入海异象被压住，树叶落入灵土。
- **神州伏笔**：大宗记录刘枫，大势力开始注意他。

## 写作禁忌

- 不要让神州人全是蠢坏傲慢，宋清河必须证明神州也有真正的秩序。
- 不要让苏影过早暴露苏凌身份，最多露出魔道手段。
- 不要让刘枫结丹太张扬，异象必须被压住。
- 不要让旧海灯解释太多大玄历史，只给残音和功能碎片。
- 不要让刘枫一结丹就被大宗收走，他仍然选择独行。
- 不要让裴知澜重复第三卷的傲慢，他在第四卷应有变化。
