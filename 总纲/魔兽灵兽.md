# 妖兽与灵兽

> 《道果》的妖兽/灵兽骨架。
> 本文只定分类、阶位、地理生态、契约规则、主灵宠方向。具体如何遭遇、如何战斗、如何取得兽材，留到正文与卷纲中处理。

---

## 一句话原则

妖兽是天地野性，灵兽是可同行的野性。  
低阶妖兽是资源与危险，高阶妖族是隐藏文明。

刘枫与灵兽的关系不走“收集宠物”路线，而走：

> **晚出场、少契约、重陪伴；一只树灵，陪他从神州初见走到终局。**

---

## 基础称呼

本书统一使用：

- **妖兽**：野外、秘境、海域中的灵性兽类，未必能契约。
- **灵兽**：可被修士驯养、契约、同行的兽类。
- **妖族**：高阶妖兽开灵智后形成的族群/文明，不等同于普通妖兽。

### 用词边界

- “魔兽”不作为正式通用称呼。
- “凶兽”可作为特殊称呼，指失去灵智、只剩杀戮本能的妖兽。
- “古兽”可指上古遗留血脉，通常出现在西极、九幽或古秘境。

---

## 阶位体系

妖兽/灵兽前期用数字阶位，后期用称号增强气势。

| 妖兽阶位 | 对应修士境界 | 说明 |
| -------- | ------------ | ---- |
| **一阶** | 炼气初中期 | 低阶妖兽，普通修士可猎杀 |
| **二阶** | 炼气后期 / 筑基初 | 开始产出有价值兽材 |
| **三阶** | 筑基 | 小宗门任务常见目标 |
| **四阶** | 金丹初中期 | 可占岛、占山、占一片海域 |
| **五阶** | 金丹后期 / 元婴初 | 一方妖主，低阶灵智明显 |
| **六阶** | 元婴 | 可称大妖，部分可短暂化形 |
| **七阶** | 化神 | 妖王层次，能与大宗长老平起平坐 |
| **八阶** | 炼虚 / 合体 | 大妖王，牵动一域局势 |
| **九阶** | 大乘 / 渡劫 | 妖圣胚子，极少现世 |
| **妖王** | 化神以上称号 | 不是固定阶位，而是地位称号 |
| **妖圣** | 大乘 / 渡劫级称号 | 传说级存在，日常不出场 |

### 灵智规则

灵智不只看阶位，也看血脉：

- 普通妖兽：三阶前多凭本能，五阶后灵智明显，六阶后可与人交流。
- 上古血脉：幼年即聪慧，可能一二阶就能理解人言。
- 妖族嫡血：出生即有族群教育，不按普通妖兽规则。
- 化形极难：六阶可短暂化形，七阶以上才较稳定。

> 写作原则：不要让低阶妖兽普遍像人一样聪明。只有特殊血脉才例外。

## 地域生态

《道果》的妖兽生态与地理强绑定。不同板块的妖兽必须有不同质感。

### 一、东海：海兽与潮兽

东海是刘枫的出生地，前 1-3 卷主要妖兽来源。

| 名称 | 阶位范围 | 特点 | 用途 |
| ---- | -------- | ---- | ---- |
| **青鳞潮鱼** | 一阶 | 群游，鳞片含微弱水灵气 | 炼气任务、低阶符材 |
| **礁甲蟹** | 一至二阶 | 甲壳坚硬，藏于礁石 | 阵盘外壳、低阶护甲 |
| **黑背海蛇** | 二至三阶 | 毒牙、善伏击、夜间上岸 | 毒材、第一卷危机可用 |
| **潮音贝** | 二至四阶 | 能扰乱神识，藏于海底 | 幻阵材料、神识法引子 |
| **裂潮鲸** | 四至六阶 | 体型巨大，能掀潮破舟 | 东海大事件级妖兽 |

东海妖兽特点：

- 多与潮汐、海雾、暗礁、海底灵脉有关。
- 战斗环境复杂：船上、礁石、海底、暴风雨。
- 前期主要作为任务目标与资源来源；中期逐渐牵出海族。

### 二、南瘴：毒妖与蛊兽

南瘴妖兽与魔道、蛊术、瘴气强绑定。

| 名称 | 阶位范围 | 特点 | 用途 |
| ---- | -------- | ---- | ---- |
| **瘴纹蜥** | 二至四阶 | 皮肤能吐瘴，善伪装 | 毒阵、隐匿材料 |
| **血藤猿** | 三至五阶 | 与血藤共生，肉身强横 | 体修材料、木毒双属 |
| **百眼蛾** | 三至六阶 | 翅上眼纹扰神识 | 幻阵、神识攻击 |
| **蛊腹蟾** | 四至六阶 | 腹中养蛊，鸣声致幻 | 魔道蛊修核心材料 |
| **青瘴蛟** | 六阶以上 | 南瘴深处妖王血脉 | 中后期强敌或盟友 |

南瘴妖兽特点：

- 毒、蛊、寄生、共生。
- 适合写“错误的生长”：生命力很旺，但不干净。
- 可作为刘枫木属路线的反面教材：同样是木，南瘴偏毒与寄生，刘枫偏生息与阵纹。

### 三、北苍：荒兽与寒兽

北苍不是主舞台，但作为对照组要有鲜明生态。

| 名称 | 阶位范围 | 特点 | 用途 |
| ---- | -------- | ---- | ---- |
| **霜角鹿** | 二至四阶 | 鹿角凝霜，感知灵脉 | 寻灵、冰属材料 |
| **雪脊狼** | 三至五阶 | 群猎，耐寒，凶狠 | 皮毛、兽牙、追踪戏 |
| **寒骨熊** | 四至六阶 | 防御极强，冬眠蓄灵 | 体修材料、肉身压迫 |

北苍妖兽特点：

- 肉身强、耐寒、群猎。
- 适合作为刘枫“阵法布局”与“硬碰硬体修”的对照。

### 四、西极：古兽与遗种

西极是上古伏笔之地，妖兽必须更古老、更稀少。

| 名称 | 阶位范围 | 特点 | 用途 |
| ---- | -------- | ---- | ---- |
| **沙眠蜃** | 四至七阶 | 制造海市蜃楼般幻境 | 上古秘境、幻阵材料 |
| **断角犀** | 五至七阶 | 角上有上古阵纹残痕 | 阵法伏笔、古战场线索 |
| **白骨鸢** | 五至八阶 | 栖于古战场，食阴煞 | 九幽伏笔、鬼修材料 |

西极妖兽特点：

- 数量少，但每一只都有“历史感”。
- 不只是怪物，而是上古大战留下的活化石。
- 第 7-8 卷用于揭开大玄天朝与原主暗线。

### 五、九幽地渊：阴兽与鬼兽

九幽地渊后期出现，不能前期滥用。

| 名称 | 阶位范围 | 特点 | 用途 |
| ---- | -------- | ---- | ---- |
| **阴灯虫** | 三至五阶 | 群聚如灯，可引魂 | 鬼修引路、九幽氛围 |
| **无面狐** | 五至七阶 | 善幻化故人面容 | 道心考验 |
| **噬魄螭** | 七阶以上 | 吞魂噬魄，近妖王 | 后期强敌 |

九幽妖兽特点：

- 不以肉身压迫为主，而以神识、魂魄、记忆压迫为主。
- 与“原主复苏”线强相关。

### 六、神州：宗门驯养灵兽

神州野外妖兽不一定最强，但灵兽驯养体系最成熟。

| 名称 | 阶位范围 | 特点 | 用途 |
| ---- | -------- | ---- | ---- |
| **云鹤** | 一至三阶 | 飞行、传讯、代步 | 大宗常见灵兽 |
| **灵狸** | 一至二阶 | 寻药、警戒 | 低阶弟子任务 |
| **护山青狮** | 三至六阶 | 守山、镇门 | 宗门威仪 |

神州特点：

- 灵兽更多作为宗门制度的一部分。
- 大宗门有灵兽园、御兽堂、契约法。
- 刘枫第 4 卷登陆神州后，才真正见识“成体系的灵兽驯养”。

## 高阶妖族文明

高阶妖族不是“野怪”，而是隐藏但强大的文明。

### 基本设定

- 妖族有自己的族谱、祖地、禁地、传承。
- 人族主流宗门并不完全承认妖族文明，只把他们称为“妖域势力”。
- 妖族与人族有过古老盟约，也有过大规模战争。
- 妖族知道一些人族宗门不知道的上古真相，尤其是**大玄天朝覆灭前后的事**。

### 妖族与大玄天朝

- 上古大玄天朝鼎盛时，曾强行册封部分妖族为“山海臣属”。
- 大玄天朝覆灭后，妖族获得喘息，但也失去部分上古传承。
- 有些妖族认为大玄天朝是暴政，有些则认为它是唯一能压住天下大劫的秩序。
- 刘枫后期会发现：妖族对“空间原主”的称呼，和人族完全不同——他们不称其为天尊，而称其为**树下旧君**。

> 这条线不要前期展开。第 6 卷南瘴、第 8 卷西极以后再逐渐显露。

## 契约灵兽规则

本书灵兽契约由宗门垄断，不是普通修士随便能学。

### 契约来源

- 成熟契约法主要掌握在神州大宗、御兽堂、少数妖族盟约家族手中。
- 东海小宗门只有粗浅驯养法，没有稳定契约法。
- 海间宗能驯海兽、养传讯灵禽，但不掌握高阶契约。

### 契约代价

- 契约会占用修士神识。
- 灵兽受伤，主人会有轻微反噬。
- 主人若强行役使灵兽赴死，会伤及道心。
- 高阶灵兽不靠契约压服，更多靠恩义、利益、共道同行。

### 契约数量

- 炼气 / 筑基：不建议契约正式灵兽。
- 金丹：可稳定契约一只低中阶灵兽。
- 元婴以上：可多契约，但神识负担明显。
- 刘枫路线：**不走多宠流**，只与一只树灵长期同行。

## 刘枫的主灵宠：小树（暂称）

> 这是中后期才出现的主灵宠，不参与前期剧情，避免抢走无名古树与刘枫自身成长线。

### 基本设定

- **形态**：树灵幼体。
- **出场**：第 4 卷，刘枫首次渡海登陆神州后，在神州古林中遇见。
- **称呼**：前期无正式名字，刘枫只叫它“**小树**”。
- **正式名字**：第 7 卷以后可取名**离枝**，象征它从古树命运中分离出来。
- **性格**：幼体老成。明明刚诞生不久，却像一棵活了很久的树。
- **表达方式**：少言，或以树叶、枝条、气息表达；开口时常说很轻的话，像在重复某种古老本能。

### 与无名古树的关系

- 小树不是无名古树的孩子，也不是古树直接产物。
- 它与无名古树存在**共鸣**：
  - 能感知刘枫身上那股“树”的气息。
  - 靠近刘枫识海空间时，会本能安静。
  - 它有时会看向刘枫身后，像看见什么刘枫看不见的东西。
- 这条共鸣只作伏笔，不要前期解释。

### 能力方向

| 能力 | 说明 |
| ---- | ---- |
| **寻根** | 感知地脉、灵脉、古树残根 |
| **护阵** | 能帮刘枫稳住阵眼，修补阵纹 |
| **木遁** | 在林木、藤蔓、草木间短距移动 |
| **听叶** | 通过草木微动感知敌人靠近 |
| **生机回流** | 轻微治疗，但不能逆天救命 |

### 战力边界

- 小树不是打手型灵宠。
- 前期更多是感知、阵法辅助、陪伴。
- 不能替刘枫解决关键战斗。
- 不能复活死人，不能外传无名古树的果实，不能直接破解原主暗线。

### 情感作用

- 第 4 卷：刘枫第一次离开东海，进入神州大陆；在神州古林中误入春生谷采药地，遇见被古林根系困住的小树灵。
- 第 5 卷：玩伴之死后，小树不安地贴近刘枫，却什么也说不出。
- 第 7-8 卷：小树对西极古战场产生恐惧，暗示它的血脉里藏着上古记忆。
- 第 9-10 卷：当无名古树与原主暗线完全浮出，小树会成为刘枫判断“树是否一定是原主”的对照。

## 海族定位

- 海族作为古老海中文明中等展开，不等同普通妖族，也不完全归入人族敌对势力。
- 第 3 卷通过海翁露出一角：旧盟石、海族信物、人族与海族古老盟约。
- 后期仍保持神秘，不让海族变成完整政治主线。
- 海族知道部分大玄旧盟真相，但不会主动为刘枫解释。

---

## 古木系灵物

- 世间存在与无名古树同类的“古木系”灵物，但极罕见。
- 第 8 卷断仙沙海的枯死巨树桩证明：无名古树不是孤例。
- 同类古木不等于都有空间、果实或原主残存；每一棵都必须绑定上古旧事与巨大代价。

## 妖兽材料与修行用途

妖兽材料不是随便爆装备，要与修行体系挂钩。

| 材料 | 用途 |
| ---- | ---- |
| **妖丹** | 修炼、炼药、阵眼、交易硬通货 |
| **兽骨** | 阵盘、法器、船骨、符笔 |
| **鳞甲** | 护甲、阵盘外壳、防御符 |
| **兽血** | 画阵、炼体、禁法 |
| **兽魂** | 幻阵、鬼修、神识法 |
| **兽角 / 兽牙** | 攻击法器、阵眼、符器 |

### 材料原则

- 低阶妖兽材料服务前期任务。
- 中阶妖兽材料服务阵法与法器。
- 高阶妖兽材料不能轻易完整取得，必须有大代价。
- 妖丹不能无限吃，乱服会妖气入体、道基驳杂。

## 写作禁忌

- 不要让刘枫前期就拥有强力灵宠。
- 不要写成“抓一只、收一只”的宠物流。
- 不要让小树替代无名古树成为新金手指。
- 不要让妖兽只当经验包；高阶妖族必须有立场、有历史、有代价。
- 不要让契约灵兽太廉价，契约必须有神识负担和道心责任。
- 不要让所有地区妖兽都一样：东海要有海味，南瘴要有毒瘴，西极要有古意，九幽要有魂魄压迫。
