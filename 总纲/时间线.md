# 时间线

> 上古 / 远古 / 近古 / 今世，先立骨架，再补血肉。
> **原则**：所有"古老传说"必须有时间锚点，避免后期前后矛盾。

## 纪元划分（建议）

- **创世纪**：天地初开、大道初分。
- **上古**：神话时代，仙神/人妖大战、天地破碎等设定。
- **远古**：法则衰退、仙路断绝（或未断）的关键节点。
- **近古**：现行宗门、王朝的奠基时期。
- **今世**：故事正文发生的当代。

## 关键设定锁

- [ ] 仙路是否断绝？何时断的？谁断的？
- [ ] 现存最古老的存在是谁？年龄多少？仍活跃否？
- [ ] 每个大势力的开宗时间与开宗祖师。
- [ ] 主角时代距上一个"大变"过去多久（决定遗迹密度与机缘浓度）。
