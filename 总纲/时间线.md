# 时间线

> 上古 / 远古 / 近古 / 今世，先立骨架，再补血肉。
> **原则**：所有"古老传说"必须有时间锚点，避免后期前后矛盾。

## 纪元划分（建议）

- **创世纪**：天地初开、大道初分。
- **上古**：神话时代，仙神/人妖大战、天地破碎等设定。
- **远古**：法则衰退、仙路断绝（或未断）的关键节点。
- **近古**：现行宗门、王朝的奠基时期。
- **今世**：故事正文发生的当代。

> 时间锁：主角时代距上一次“大变”不可考，只知“大玄亡后”。具体年份被五圣门、九州大宗和旧禁制共同抹成断代。

## 大事件骨架

| 纪元 | 距今 | 事件 | 影响 / 留下的遗产 |
| ---- | ---- | ---- | ----------------- |
| 上古 | 不可考 | 大玄天朝统一修真界，玄都帝君立朝，渡生天尊、镇西侯等名号留于残史 | 天下第一次被统一秩序压住，也留下镇劫阵体系 |
| 上古末 / 远古初 | 不可考 | 大玄覆灭，旧都一半陷入九幽，仙路“断绝”的说法开始流传 | 五圣门与九州十六派格局逐渐形成 |
| 近古 | 数百至千年尺度 | 海间宗开宗，刘家祖上刘青崖成金丹又坐化 | 东海小宗门、小世家格局定型 |
| 今世 | 正文当代 | 刘枫出生于青礁岛刘家 | 无名古树再度等到“树下后来者” |
