# 宗门与王朝

> 《道果》势力体系。
> **设计原则**：用最少的"档位"撑起整个修仙界；只详写主角真正接触的，其它留待主角走到再补。

---

## 宗门五档梯队

按"宗主境界 / 影响范围"分档，**主角一生从最底档起步，跨过其中四档**：

| 档次 | 名称 | 宗主境界 | 数量 | 主角接触卷次 |
| ---- | ---- | -------- | ---- | -------------- |
| **第一档：天下圣地** | 「**五圣门**」 | 大乘 / 渡劫 | 5 家 | 卷 8 远观 / 卷 9 真正接触 |
| **第二档：神州大宗** | 「**九州十六派**」 | 化神 / 炼虚 | 16 家（神州为主） | 卷 4 远景 / 卷 6-7 正式接触 |
| **第三档：地方一流宗门** | （各域 2-5 家） | 元婴 / 金丹后期 | 数十家 | 卷 3-4 |
| **第四档：地方二三流宗门** | （数百家） | 金丹 / 筑基圆满 | 数百家 | **主角入此档（卷 1-3）** |
| **第五档：修仙世家** | （数千家） | 筑基 / 炼气祖辈 | 数千家 | **主角出身（卷 1）** |

> 跨档原则：每跨一档都意味着一次境界跃迁与一次世界观升级——读者不会信息过载，每卷都觉得"哦原来上面还有更高的"。

## 第二档：九州十六派（神州主流宗门，详细名号留待）

- 神州九州，每州 1-3 家入此档，共十六家。
- 化神 / 炼虚级宗主；元婴长老数十。
- 第 4 卷主角首次渡海登陆神州时，**远远看见某州大宗的山门**——这是开眼界的第一画面。
- 第 7 卷主角正式跻身神州道统，开始与十六派打交道。

## 第三档：地方一流宗门（数十家）

- 元婴 / 金丹后期宗主；一州或一海域的"老大"。
- 主角在第 3-4 卷接触；通过出色表现引起其注意。
- 例：东海的一流宗门待定（暂名「**碧涛宗**」或留待）。

## 第四档：地方二三流宗门（主角入门处）

### **海间宗**（主角第 1-3 卷主舞台）

> 主角刘枫第 1 卷外门考核入此宗。

- **位置**：东海·万岛海·距青礁岛约百里的「**海间岛**」上。
- **档次**：东海一带颇有名气的二三流宗门；宗主为**筑基圆满**，距金丹仅一线之隔。
- **传承**：约八百年，由一位金丹散修开宗，后金丹陨落于一次海族入侵，宗门自此再未出过金丹。
- **山门风貌**：依海岛主峰而建；殿宇并不宏伟但极有海上风骨；常年海风，朱漆门匾被晒得发白。
- **修行风格**：
  - 主修水属功法、御舟之术。
  - 弟子日常需"出海历练"——和神州山林宗门完全不同的修行节奏。
- **门人结构**：
  - **核心弟子**：约二十人，金丹长老亲传。
  - **内门弟子**：约二百人，筑基期，宗内主力。
  - **外门弟子**：约千人，炼气期，多为周边岛屿世家送来的子弟。
  - **杂役弟子**：数千人，凡人或低品灵根。
- **派系**：内有"老人派"（守海间岛本土）与"新进派"（主张登陆神州、谋求向上跃迁）的暗斗。
  - 这条暗斗会成为主角第 2-3 卷的副线。
- **与刘家的关系**：
  - 海间宗与万岛海各大世家有"送子入门"的传统，**刘家庶子常被送来此宗的外门**。
  - 刘家是海间宗的"小金主"之一——但地位并不高。
- **写作意义**：
  - 给主角一个**真实的"小宗门"环境**，避免一开始就"天才入大派"的爽文俗套。
  - 海间宗的"老/新两派暗斗" = 让主角在第 2-3 卷有内部冲突可写。
  - 主角后期回望，海间宗将是他**第一段真正的修行起点**——情感锚很重。

## 第五档：修仙世家（主角出身）

### **刘家**（主角出身）

> 主角刘枫的出身——青礁岛南端的小修仙世家。

- **位置**：东海·万岛海·**青礁岛**南端。
- **规模**：占据青礁岛南端约四分之一的灵地，下辖三个凡人渔村。
- **族中境界**：
  - 现任族长：**筑基中期**（刘枫的伯父，掌族二十年）。
  - 长老：约 5 位筑基，10 余位炼气后期。
  - 嫡系子弟：30-50 人，多在外门与海间宗任职。
  - 庶系子弟：百人左右，地位极低。
- **历史**：
  - 三百年前曾出过一位**金丹真人**——刘家由此立家。
  - 金丹真人坐化后，刘家再未出过金丹，逐渐衰落，如今只是青礁岛上"一户还过得去的世家"。
  - 族中流传"祖上有福泽，后人未必能享"——一句被嫡系当套话、被庶系当怨言的祖训。
- **主角处境**：
  - 父：刘家旁支子弟，常年在外为家族办事，几乎不归。
  - 母：青礁岛凡人渔村出身，被纳为侧室；地位极低。
  - 主角自身：庶子，无灵根优势，自小被嫡系压制，资源稀薄。
- **关键人物（待具体设定）**：
  - 母亲（**温情锚**）：朴素温柔，一生最大的愿望是"枫儿能活下去"。
  - 一位旁支老叔伯（**授业锚**）：被族中嫡系排挤的老人，对主角态度奇好——他可能知道一些关于主角身世的事，但不肯说。
  - 一位嫡系堂兄/族兄（**第一仇人**）：将在第 1 卷下狠手，导致主角濒死被引入空间。
- **写作意义**：
  - 提供一个"**地位低 + 资源少 + 还不至于完全绝路**"的标准修仙文起点。
  - 嫡系压制 + 衰落世家 + 祖训暗示 = 主角早期张力的全部来源。
  - 第 1 卷末：主角通过外门考核入海间宗——**与刘家正式拉开距离**，但**不彻底决裂**。
  - 中后期：刘家几乎被遗忘；最后某一卷主角衣锦还乡，**整个青礁岛已物是人非**。

## 王朝层（轻量化处理）

> 修仙文里王朝最容易写崩——写多了喧宾夺主。本书王朝层只起"凡俗背景"作用。

### 凡俗王朝（不主导剧情）

- 修真界主体由宗门主导，王朝只是"凡俗管理者"，地位远低于宗门。
- 各域有各自的凡俗王朝，与本地宗门保持微妙的"挂名"关系。
- 主角一生很少与王朝直接打交道——除非剧情需要。
- 暂列：北苍有「**苍北王朝**」、南瘴有「**百越蛮国**」、神州有数个王朝（待定）。

### 上古王朝：「**大玄天朝**」（全书第二大伏笔）

> 仅次于"无名古树"的大伏笔。

- **历史**：
  - 上古时代曾有一个**统一整个修仙界**的修真王朝——大玄天朝。
  - 由数位渡劫真君与大乘修士联合开创，统御天下宗门数千年。
  - 鼎盛时**仙凡同朝**，是修仙史上唯一一次"统一秩序"。
- **覆灭**：
  - 一夜之间土崩瓦解；具体原因不可考。
  - **覆灭与一位渡劫失败者有关**——这位失败者，正是空间原主。
  - 大玄天朝的覆灭，让修仙界自此分裂为如今的"五圣门 + 九州十六派"格局。
- **遗存**：
  - **西极的上古战场**——大玄天朝最后一战之地。
  - **神州深处的旧都**——封禁千年，无人能入。
  - **九幽地渊深处**——传说有大玄天朝最后的遗诏。
  - **某些上古传承功法** = 大玄天朝时代的产物。
- **戏份**：
  - 第 1 卷：主角偶尔听见族中老人提起"上古曾有大玄天朝"，**当作神话听**。
  - 第 4-6 卷：主角逐步接触到大玄天朝时代留下的**功法残卷与古迹**。
  - 第 7-8 卷：主角进入西极，**逐步揭开大玄天朝覆灭真相**——并第一次窥见"原主"的轮廓。
  - 第 9-10 卷：原主复苏，主角终于明白：**大玄天朝是被空间原主一手覆灭的**。
- **写作要点**：
  - 大玄天朝的设定**永远只在传说与残卷中出现**，**不要让任何一位"亲历者"以"长辈姿态"复述历史**——一旦有人复述，神秘感全失。
  - 让主角和读者一起，**从碎片里拼出真相**。
