# 境界体系

> 与 `00_创作约束.md` 中"境界体系"一节配套使用。
> 本文件为可填充骨架：每境界先把**能力边界**和**突破门槛**写死，再写细节。

## 总览

炼气 → 筑基 → 金丹 → 元婴 → 化神 → 炼虚 → 合体 → 大乘 → 渡劫 → 仙人

- **质变节点**（推荐）：筑基、金丹、元婴、化神、渡劫——每跨一节寿元/神识/手段必须显著跃升。
- **小阶建议**：每境界分 初期 / 中期 / 后期 / 圆满（巅峰），用于战力差结算。

## 小阶战力差

- 同一大境界内，小阶差距明显但不是绝对碾压。
- 初期 → 中期：灵力量、术法熟练与神识精度约强三成。
- 中期 → 后期：可形成稳定压制，但仍可能被法宝、阵法、环境翻盘。
- 后期 → 圆满：差的是“破境资格”，不只是灵力多少；圆满修士可短暂压住同境后期多人。
- 跨大境界：通常不可硬拼。刘枫的越阶胜利必须依靠提前布阵、地利、果实代价、敌人误判，不能正面无代价硬砍。

## 伪境界与跌境

- **伪境界**：可存在，如药力强提、血祭借力、残魂附体、阵法灌顶。
- 伪境界只能获得部分战力，不获得完整寿元、神识和规则理解。
- **跌境机制**：可逆但代价极大，通常需要同阶以上灵物、长时间闭关和心境修补。
- 跌境后强行再战，会造成道基裂痕，影响后续突破。

## 借力突破

- 借力突破合法但有账：宗门灌顶、双修借力、血祭、香火、妖丹、古树果都能算“外力”。
- 正道借力重契约和后续偿还；魔道借力重即时收益和反噬。
- 双修不作为主角主线手段，若出现必须有心境、因果和双方损耗，不作轻浮爽点。

## 仙界登场原则

- 仙界不作为日常舞台。
- “仙路断绝”是修真界通行说法，但后期会发现这不完全是真相。
- 第 10 卷可以触及九霄云海与飞升真相，但不写成换地图开新篇。

## 通用维度（每境界都要写明）

- 寿元：自然寿命上限。
- 神识：覆盖范围与精度。
- 飞遁：能否御物、速度量级。
- 修炼方式：吸纳灵气 / 凝练丹田 / 神魂蜕变 / 法则感悟……
- 战力上限：能压制谁、被谁碾压。
- 权限：可炼何种丹药/法器/阵法、可入何种秘境。
- 突破门槛：资源 + 功法 + 心境 + 天劫（如有）。
- 失败代价：走火入魔 / 道基受损 / 寿元折损 / 跌境。

---

## 一、炼气

- 定位：入门，凡人筑骨洗髓。
- 寿元：~120 年。
- 神识：尚未凝聚，仅"灵觉"。
- 飞遁：不能御物飞行，至多御剑短距。
- 战力：常人难敌；十名甲士可围杀。
- 突破筑基门槛：筑基丹 / 灵脉洞府 / 心境无杂。

## 二、筑基

- 定位：正式踏入修仙，寿命跨越凡人极限。
- 寿元：~200 年。
- 神识：初步凝聚，丈许范围。
- 飞遁：可御物短途飞行。
- 战力：一人压一寨；筑基后期可硬抗低阶法器。
- 突破金丹门槛：道基稳固 + 灵气凝液 + 一线生机。

## 三、金丹

- 定位：地仙之资，宗门骨干。
- 寿元：~500 年。
- 神识：里许，可分神操控法器。
- 飞遁：御剑/御舟/驾云。
- 战力：一郡之主级别；金丹圆满可硬撼元婴一招。
- 突破元婴门槛：金丹温养 + 元神孕育 + 心魔劫。

## 四、元婴

- 定位：一方霸主，宗门长老级。
- 寿元：~1000 年。
- 神识：十里以上，可元神出窍。
- 飞遁：长途遁光，速度成倍提升。
- 战力：可灭一国；元婴期始可独立开辟洞府。
- 突破化神门槛：元婴蜕凡 + 法则初触 + 雷劫（小三灾）。

## 五、化神

- 定位：触碰天地法则，宗主级。
- 寿元：~2000 年。
- 神识：百里，可凝实化形。
- 飞遁：瞬移短距 / 缩地成寸。
- 战力：跺脚一州震动；化神坐镇即一宗根本。
- 突破炼虚门槛：法则雏形 + 心境圆融 + 天地认可。

## 六、炼虚

- 定位：以神驭虚，半步陆地真仙。
- 寿元：~5000 年。
- 神识：千里。
- 飞遁：法则短瞬，跨域不易。
- 战力：单挑一宗有余；可独镇一域。
- 突破合体门槛：神魂与天地共鸣 + 道果初凝。

## 七、合体

- 定位：神形合一，大派太上。
- 寿元：~10000 年。
- 神识：跨域感应。
- 战力：合体出手即抹城；少有同辈能制。
- 突破大乘门槛：道果稳固 + 与道契合 + 因果不乱。

## 八、大乘

- 定位：人间至尊，距仙一步。
- 寿元：~30000 年（具体看世界设定）。
- 战力：可推动小型天地法则；大乘之争波及万里。
- 突破渡劫门槛：道果圆满 + 因果偿清 + 接引天劫。

## 九、渡劫

- 定位：与天博命，飞升前夕。
- 寿元：临近天数，必须渡劫续命。
- 战力：渡劫之争即天地浩劫。
- 关键设定：劫数次数、劫种（雷/火/风/心/业）、可否他人助劫。
- 渡劫成败：成则飞升或留世为仙；败则形神俱灭，福泽后人或祸延宗门。

## 十、仙人

- 定位：超脱本界，规则之上。
- 关键设定：是否真有仙界？飞升后是否还能返回？仙凡界限如何隔绝？
- 注意：仙人入场即"上限破坏者"，**慎用**；最好留作终局/背景，不作日常战力。
