# 宝物装备

> 《道果》的宝物装备骨架。
> 本文只定品阶、分类、经济流通、主角本命物与禁忌。具体宝物如何获得、如何修复、如何在战斗中发挥作用，留到正文和卷纲中处理。

---

## 一句话原则

宝物不是“随手掏出来的外挂”，而是资源、势力和代价的结晶。  
低阶宝物可交易，高阶宝物被宗门和大势力垄断。

刘枫的装备路线：

> **前期靠普通阵盘、阵旗、符箓保命；中期靠准备和布阵取胜；第 7 卷后才获得本命阵盘「年轮盘」。**

---

## 品阶体系

宝物装备采用传统器物阶，后期与“道/仙”接轨。

| 品阶 | 对应境界 | 说明 |
| ---- | -------- | ---- |
| **法器** | 炼气 / 筑基初 | 低阶修士常用，坊市可买卖 |
| **灵器** | 筑基 / 金丹初 | 二三流宗门核心弟子、筑基修士常用 |
| **法宝** | 金丹 / 元婴 | 金丹真人本命物层级，地方宗门镇门之物 |
| **灵宝** | 元婴 / 化神 | 大宗门长老级宝物，极少流入坊市 |
| **道宝** | 炼虚 / 合体 / 大乘 | 与大道法则相连，五圣门级传承 |
| **仙器** | 仙人 | 只作传说或终局伏笔，不在日常剧情滥用 |

### 副品

每一阶只分三品：

- 下品：可用。
- 中品：完整。
- 上品：同阶珍贵。

> 禁止再堆“极品、伪灵宝、半步道宝”等太多中间层，避免后期战力膨胀。

---

## 宝物分类

本书宝物只分五大类，保持简洁。

| 分类 | 作用 | 常见例子 |
| ---- | ---- | -------- |
| **攻伐类** | 杀敌、破阵、远攻近战 | 飞剑、法珠、兽牙刃 |
| **防御类** | 护身、挡劫、保命 | 护甲、玉佩、龟甲盾 |
| **飞行类** | 代步、逃生、远行 | 飞舟、御风旗、云梭 |
| **储物类** | 携带物资 | 储物袋、储物戒 |
| **阵器类** | 布阵、控阵、破阵 | 阵盘、阵旗、阵眼器 |

### 主角侧重点

- 刘枫不走“满身神装”路线。
- 他的核心装备是**阵器**。
- 攻伐、防御、飞行类宝物只服务于剧情需要，不长期堆叠。

---

## 储物装备

### 基本规则

- **低阶储物袋常见**：炼气后期、筑基修士多可拥有。
- **高阶储物戒稀少**：通常由金丹以上修士或大宗核心弟子持有。
- **大型储物法宝极罕见**：大宗门、商会、军阵才有。

### 与刘枫识海空间的边界

- 刘枫仍需使用普通储物袋，避免外人起疑。
- 识海空间不是普通储物空间，不可被当成随身仓库滥用。
- 无名古树空间内可保存果实，但不代表能随意堆放外界杂物。
- 若将大量外界之物带入识海空间，会扰动灵土气息，增加被古树/原主感知的风险。

> 这样既保留储物袋的修仙味，也避免刘枫的识海空间变成万能仓库。

## 消耗品体系

低阶修士斗法不只靠功法，消耗品很重要，但不能决定一切。

### 符箓

- 存在感：**中等**。
- 炼气 / 筑基斗法常用，可补短板、制造转机。
- 符箓可以保命，但不能替代修为和准备。
- 高阶符箓价格昂贵，宗门和坊市限量流通。

### 丹药

- 丹药作为背景资源存在：刘枫会买、换、抢，但**不炼丹**。
- 低阶丹药可在坊市流通。
- 高阶丹药被宗门垄断。
- 丹药不能无脑连吃，滥服会丹毒积累、道基虚浮。

### 阵盘与阵旗

- 刘枫前中期最重要的消耗品。
- 阵盘可重复使用，但会磨损。
- 阵旗多为一次性或半消耗品。
- 布阵需要时间、地点、材料、阵眼。

### 灵石

- 灵石是修真界通用货币，也是阵法和飞舟的燃料。
- 低阶灵石用于修炼、交易、阵法消耗。
- 高阶灵石被宗门和商会掌控。

---

## 宝物经济

### 总体格局

本书宝物经济以**宗门垄断**为主。

- 低阶法器、符箓、丹药可在坊市交易。
- 灵器以上的成套装备，大多由宗门、世家、商会控制。
- 法宝以上很少公开买卖，通常绑定身份、师承、宗门功绩。
- 道宝不进入交易体系，只能继承、镇宗、夺取或承认其主。

### 东海

- 东海坊市流通：海兽材料、低阶法器、船器、储物袋、符箓。
- 飞舟、避水珠、御潮旗等水行宝物较常见。
- 高阶海兽材料被大宗门和海商垄断。

### 神州

- 神州大宗掌握成熟炼器体系。
- 大宗弟子装备明显优于地方修士。
- 刘枫第 4 卷登陆神州后，会第一次真正感受到“装备差距”。

### 南瘴 / 西极 / 九幽

- 南瘴多毒器、蛊器、血器。
- 西极多古宝残片、断裂阵器。
- 九幽多魂器、阴器、镇魂物。

## 刘枫的装备路线

| 阶段 | 境界 | 装备状态 | 核心特点 |
| ---- | ---- | -------- | -------- |
| 前期 | 炼气 → 筑基 | 普通储物袋、低阶阵旗、基础符箓 | 穷、谨慎、靠准备 |
| 中期 | 金丹 → 元婴 | 自制/购置阵盘、临时阵器、少量护身物 | 以阵法弥补正面战力不足 |
| 后期 | 化神 → 炼虚 | 获得本命阵盘「年轮盘」 | 木生阵真正成型 |
| 终局 | 合体 → 渡劫 | 年轮盘与自身道法合一 | 天地为阵，决定是否斩断古树之路 |

## 本命阵盘：「年轮盘」

> 第 7 卷后出现，刘枫真正意义上的本命装备。

### 基本定位

- 类型：**木质阵盘**。
- 品阶初显：法宝 / 灵宝之间（具体以剧情落点为准）。
- 出场卷次：第 7 卷「上古回响」。
- 来源：某次上古遗迹或旧阵残境中偶然获得。
- 与无名古树关系：**有共鸣**，材质未知，能与无名古树共振，但不是无名古树的枝。

### 名称取意

「年轮」对应三层含义：

1. **木**：阵盘为古木所成，年轮如阵纹。
2. **时**：刘枫的金手指以甲子为节奏，年轮记录岁月。
3. **道**：每一道年轮都是一圈选择，刘枫一生都在“绕树而行”。

### 核心功能

- 作为阵法核心、阵眼、阵盘使用。
- 能承载刘枫的木属灵力，让阵纹像根须一样生长。
- 可稳定大型阵法，减少布阵反噬。
- 对古阵、残阵、木属阵纹有特殊感应。

### 边界

- 年轮盘不是攻击法宝。
- 不能替刘枫即时秒杀强敌。
- 布阵仍需要时间、材料、地利、阵眼。
- 共鸣无名古树时，会让刘枫更接近原主气息。

### 暗线作用

- 刘枫起初以为年轮盘只是木系古阵器。
- 第 8 卷后发现：年轮盘的部分阵纹与无名古树树皮纹路相似。
- 第 9 卷后进一步发现：年轮盘可能来自另一种同类古木，或者来自大玄天朝时代某个“种阵”体系。
- 终局时，年轮盘会成为刘枫判断“树是否等于原主”的关键证据之一。

## 禁忌宝物（三类严禁滥发）

### 一、复活 / 续命类

- 必须极罕见。
- 不能轻易救回死人。
- 否则第 5 卷“玩伴之死”会崩。
- 续命宝物可存在，但只能延命，不可逆死。

### 二、空间类

- 必须极罕见。
- 低阶储物袋不算空间宝物，只是携物工具。
- 真正能藏身、开界、隔绝因果的空间宝物，不能随便出现。
- 否则无名古树空间会贬值。

### 三、时间类

- 必须极罕见。
- 不能出现随意加速修炼、倒转时间、改变甲子结果周期的宝物。
- 否则“每甲子三果”的节奏会崩。

> 三类禁忌宝物若必须出现，必须绑定大代价、大因果、大势力，不能作为普通机缘。

---

## 写作禁忌

- 不要让刘枫靠捡宝物连续升级。
- 不要让每场危机都靠新宝物解决。
- 不要让高阶宝物进入普通坊市。
- 不要让储物装备泛滥到失去资源管理意义。
- 不要让年轮盘过早出现；它必须在刘枫阵法路线成熟后才有意义。
- 不要让年轮盘成为第二个金手指；它是阵法核心，不是万能外挂。
- 不要滥发复活、空间、时间类宝物。
