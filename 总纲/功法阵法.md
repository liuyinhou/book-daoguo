# 功法阵法

> 《道果》的修行“语法”。
> 本文只定骨架：品阶、属性、功法分类、阵法分类、主角刘枫的路线。具体招式、机缘、战斗细节写到正文时再展开。

---

## 一句话原则

功法决定“修士如何吸纳天地”，阵法决定“修士如何借天地之势”。  
前者是个人之道，后者是天地之理。

刘枫的根本路线是：

> **以古树引出的木属心法为根，以阵法为辅，以隐忍布局替代正面碾压。**

---

## 品阶体系

功法、阵法、术法、法宝大体共用以下品阶，便于全书统一判断强弱。

| 品阶 | 对应境界 | 说明 |
| ---- | -------- | ---- |
| **黄阶** | 凡人 → 炼气 | 入门基础，地方世家与小宗门常见 |
| **玄阶** | 筑基 | 二三流宗门核心传承 |
| **地阶** | 金丹 | 地方一流宗门、金丹真人传承 |
| **天阶** | 元婴 → 化神 | 大宗镇山之物，轻易不传 |
| **道阶** | 炼虚 → 合体 / 大乘 | 五圣门级传承，与“道”相连 |
| **仙阶** | 仙人 | 只作传说或终局伏笔，日常剧情不滥用 |

### 副品

每阶只分三品：

- 下品：可修。
- 中品：完整。
- 上品：同阶珍贵。

> 禁止再细分“极品、绝品、半步、伪”等过多层级，避免战力体系膨胀。

## 属性体系

### 基础五行

| 属性 | 特点 | 常见修士 |
| ---- | ---- | -------- |
| **金** | 锋锐、杀伐、器物 | 剑修、器修 |
| **木** | 生息、恢复、缠绕、阵纹生长 | 灵植师、医修、阵修、刘枫主修 |
| **水** | 流动、隐匿、御舟、化形 | 东海修士、海间宗主流 |
| **火** | 爆发、炼化、破邪 | 丹师、火法修士 |
| **土** | 厚重、防御、镇压 | 体修、守阵修士 |

### 特殊属性

| 属性 | 来源 | 使用原则 |
| ---- | ---- | -------- |
| **雷** | 天劫、雷灵根、雷兽材料 | 强但稀少，常与劫数相关 |
| **阴** | 鬼修、九幽、夜行法 | 可用于隐匿、控魂、养鬼 |
| **阳** | 正道、血气、日火 | 可破邪、驱阴、护体 |
| **冰** | 水之极 | 作为独立属性出现，但根源仍属水 |
| **风** | 木之变 | 速度、身法、切割，适合高机动战斗 |
| **毒** | 木土混化 / 南瘴体系 | 与蛊、瘴、妖修相关 |
| **空** | 空间裂隙、上古禁术 | 极少出现，必须与禁忌和代价绑定 |

> 特殊属性可以存在，但不能随剧情需要无限新增。新属性必须能归入五行、雷、阴阳、冰、风、毒、空之一。

## 功法分类

| 分类 | 作用 | 举例 |
| ---- | ---- | ---- |
| **心法** | 根本修炼法，决定灵气如何入体、入丹田、化神识 | 《青木生息诀》 |
| **战诀** | 攻防招式，用于斗法 | 水箭术、缠木术、落潮剑诀 |
| **身法** | 位移、隐匿、逃生 | 浮潮步、风影遁 |
| **神识法** | 探查、感知、防夺舍 | 听潮术、定神诀 |
| **禁法** | 代价极大、短时换命 | 血遁、燃寿、替劫 |

### 功法约束

- 心法不可频繁更换，换则伤根基。
- 战诀可多学，但必须服务于心法。
- 禁法不能常用，每次都要付出明确代价。
- 高阶功法不是“更大数字”，而是**对天地规则的理解更深**。

## 主角刘枫的根本功法：《青木生息诀》

> 这是刘枫的第一门心法，也是全书最重要的功法伏笔。

### 来源

- 刘枫濒死后被引入识海空间，在无名古树下沉睡。
- 醒来后，他脑中自然多出一套陌生的行气路线。
- 这不是他自悟，也不是有人传授，而是**无名古树引导出来的功法**。
- 刘枫最初只把它当成“树下睡梦所得”，不知其来历。

### 表面品阶

- 前期表现：**黄阶上品木属心法**。
- 特点：温和、稳固、恢复快，不易走火入魔。
- 缺点：爆发不足，斗法杀力弱，修炼速度在外界并不惊人。

### 真正特性

| 特性 | 说明 |
| ---- | ---- |
| **生息** | 修复经脉、稳定道基，适合刘枫多次死里逃生 |
| **藏锋** | 灵力气息温和，不易被人看出真实底蕴 |
| **养阵** | 可让阵纹如根须般缓慢生长、修复、蔓延，后期与阵道深度结合 |
| **引树** | 每次进阶时，会与无名古树产生共鸣，推动原主气息苏醒 |

### 暗线

- 《青木生息诀》不是完整功法，而是一条“种子法”。
- 它会随刘枫境界提升，逐步显露后篇：
  - 炼气：青木生息。
  - 筑基：根须入海，灵力绵长。
  - 金丹：一木成林，阵纹生长。
  - 元婴：神识化枝，分念如叶。
  - 化神后：道果初显，开始触及无名古树原主的痕迹。
- 第 7-8 卷后，刘枫会意识到：**他修的不是某门“功法”，而是在沿着那棵树曾经走过的路生长。**

### 重大风险

- 修炼越深，刘枫与无名古树的联系越深。
- 吃果可加速《青木生息诀》，但同时推进原主复苏。
- 后期最大问题不是“功法不够强”，而是：

> **这门功法到底是在成就刘枫，还是在把刘枫长成另一个人？**

## 刘枫的修行路线

| 阶段 | 境界 | 主修 | 辅修 | 战斗风格 |
| ---- | ---- | ---- | ---- | -------- |
| 前期 | 炼气 → 筑基 | 《青木生息诀》 | 基础阵法 | 低调、防守、缠斗、借地形 |
| 中期 | 金丹 → 元婴 | 青木生息进阶篇 | 阵法布置、阵盘炼制 | 以阵困敌，以木生阵，以弱胜强 |
| 后期 | 化神 → 炼虚 | 青木化神 / 道果篇 | 大阵、封印、空间禁制 | 提前布局，战斗未起胜负已定 |
| 终局 | 合体 → 渡劫 | 自创之道 | 天地为阵 | 不再只借树，而是决定是否斩断树 |

### 为什么主角辅修阵法

- 阵法适合隐忍主角：提前布置、借势而动、不正面硬拼。
- 木属灵力适合养阵：阵纹如根须，能缓慢生长、修复、蔓延。
- 与无名古树视觉一致：树根、枝脉、阵纹三者可以互文。
- 阵法能把“种田复利感”转化为战斗力：主角越早布置，后期收益越大。

## 阵法体系

### 阵法七类

| 类型 | 作用 | 常见场景 |
| ---- | ---- | -------- |
| **聚灵阵** | 聚拢灵气，加速修炼 | 洞府、宗门、灵田 |
| **守护阵** | 防御、护山、护身 | 山门、洞府、船阵 |
| **困敌阵** | 限制行动、封锁退路 | 斗法、追杀、秘境 |
| **杀阵** | 主动攻击 | 宗门大战、秘境陷阱 |
| **幻阵** | 迷惑神识、制造假象 | 试炼、陷阱、逃生 |
| **传送阵** | 远距移动 | 大宗门、古遗迹 |
| **锁灵阵** | 封印灵力、锁物、镇压 | 禁地、镇魔、封树 |

### 阵法构件

- **阵盘**：便携核心，适合散修和低阶修士。
- **阵旗**：临时布阵，速度快但容易损毁。
- **阵眼**：阵法核心，可为灵石、法宝、灵脉、活物。
- **阵纹**：刻在地面、器物、骨骼、树根上的纹路。
- **阵基**：大型阵法的根基，多与地脉、灵脉相连。

### 阵法品阶

同功法：

- 黄阶阵：炼气可用，小范围聚灵、警戒。
- 玄阶阵：筑基可用，洞府、坊市常见。
- 地阶阵：金丹可用，可护岛、护宗。
- 天阶阵：元婴 / 化神可用，可镇一州。
- 道阶阵：炼虚以上，牵动天地法则。

### 刘枫的阵法特色：木生阵

刘枫后期阵法不同于普通阵师：

- 普通阵师：刻阵纹、插阵旗、摆阵盘。
- 刘枫：以木属灵力“种”下阵纹，让阵纹像根须一样缓慢生长。

优点：

- 隐蔽，不易被发现。
- 可长期埋伏，越早布置越强。
- 可与地脉、树根、藤蔓、灵植结合。

缺点：

- 起效慢，不适合临场强攻。
- 怕火、雷、强行破土。
- 若敌人提前识破阵眼，极易反噬。

## 宗门功法格局

### 刘家

- 刘家常规功法以水属为主，皆为黄阶下中品。
- 刘枫修《青木生息诀》与家族主流不合，因此早期更容易隐藏真实根基。
- 刘家不知《青木生息诀》来源，也不能解释刘枫修炼速度。

### 海间宗

- 海间宗主修水属法门、御舟术、近海斗法。
- 外门弟子多学基础水法。
- 刘枫入宗后表面学习水法，实际以《青木生息诀》为根。
- 海间宗有基础阵法课，这将成为刘枫正式接触阵道的入口。

### 神州大宗

- 各大宗门功法更完整，也更强调道统。
- 神州大宗会发现刘枫功法“似木非木、似阵非阵”，从而注意到他的异常。

## 特殊修法体系（存在但不抢主线）

| 体系 | 地域 | 作用 |
| ---- | ---- | ---- |
| **剑修** | 神州 / 五圣门 | 杀力最高，作为亲宗天骄或隐藏底牌出现 |
| **体修** | 北苍 / 南瘴 | 肉身强横，适合作为压迫感敌人 |
| **鬼修** | 九幽地渊 | 控魂、阴气、夺舍，与原主暗线呼应 |
| **妖修** | 南瘴 / 东海海族 | 血脉、化形、妖丹，与魔兽灵兽体系相连 |
| **魔修** | 南瘴 / 上古遗法 | 走捷径、重代价，是刘枫“自由之道”的反面教材 |

原则：

- 每种体系只在主角真正遇到时展开。
- 不提前写满所有规则。
- 每种体系都要对应一种“错误的自由”或“错误的长生”。

## 战斗结算原则

一场斗法结果由六项共同决定：

1. **境界**：基础压制。
2. **功法品阶**：同境界内的差距。
3. **属性克制**：五行、雷、阴阳、特殊属性。
4. **准备时间**：刘枫阵法路线的核心优势。
5. **环境**：海上、林中、洞府、秘境差异极大。
6. **代价**：是否动用果实、禁法、寿元、道基。

> 刘枫不是“越阶硬砍型”主角，而是“提前埋线、借势翻盘型”主角。

## 写作禁忌

- 不要让刘枫突然会很多无关功法。
- 不要让《青木生息诀》一开始就表现得像神功。
- 不要把阵法写成万能外挂：布阵需要时间、地点、材料、阵眼。
- 不要每遇到一个强敌就发明一个新属性。
- 不要把“木属”只写成治疗，木还代表生长、缠绕、寄生、根系、年轮、复苏。
- 不要让主角辅修丹器阵三项全能；本书主角辅修只走**阵法**，其余至多粗懂。
