# 人物关系

> 《道果》人物关系骨架。
> 本文只定人物功能、关系位置、出场阶段、命运走向。具体对白、恩怨细节、相遇事件，留到正文和卷纲中处理。

---

## 一句话原则

人物不是为了热闹而存在，而是为了拷问刘枫的“自由之道”。

刘枫身边的人大致分为五类：

| 类型 | 作用 |
| ---- | ---- |
| **温情锚** | 证明刘枫仍是人，不是纯粹求道机器 |
| **道心锚** | 一旦失去，会改变刘枫对“自由”的理解 |
| **仇恨锚** | 推动刘枫第一次真正出手 |
| **师门锚** | 让刘枫学会修真界的规则 |
| **镜像锚** | 让刘枫看见“另一种自己” |

---

## 关系总览

```text
                       宋清河（秩序侧镜像）
                              ↑
                              │
刘清意（启蒙授业） → 刘枫 ← 许潮（童年/道心锚）
                              │
                              ↓

刘岳（第一仇人） ── 推动刘枫濒死觉醒

无名古树 / 原主 ── 最终镜像与终局对手
```

## 主角：刘枫

- **身份**：东海·万岛海·青礁岛·刘家庶子。
- **根本命题**：求长生、求自由，但最终必须回答“自由是否一定意味着孤独”。
- **性格底色**：隐忍冷静，温良不圣母。
- **人物关系核心**：
  - 幼年缺失母亲与父亲，情感结构天然孤。
  - 许潮是童年时期少数真正同龄亲近者。
  - 刘清意是修行启蒙者，让刘枫知道“怎么活下去”。
  - 宋清河与苏凌分别代表两条错误或偏执的自由之路。

---

## 刘家早期人物

### 母亲（不提姓名）

> 难产而亡，刘枫从未见过。

- **身份**：青礁岛凡人渔村出身，被刘家纳为侧室。
- **命运**：生刘枫时难产而亡。
- **作用**：
  - 她不是活着的牵制，而是刘枫心中一块空白。
  - 刘枫对“母亲”的理解来自他人零碎描述，因此更孤。
  - 她的死亡使刘枫在刘家没有真正庇护。
- **写作边界**：
  - 正文中不主动提及母亲姓名，保持“空白感”。
  - 不要让母亲后期突然有大身世，避免抢主线。
  - 可以保留少量遗物，如一枚旧木簪、一件旧衣、几句旁人转述的话。

### 父亲（姓名待定）

- **身份**：刘家旁支子弟。
- **状态**：长期不在，只作为刘枫身份背景。
- **作用**：
  - 让刘枫的“庶子”身份成立。
  - 造成父权缺席，但不抢早期剧情。
- **写作边界**：
  - 前期不展开父亲线。
  - 若后期需要出现，必须服务刘家旧事或主角心境，不单独开大线。

### 刘清意

> 刘枫的启蒙授业者，刘家旁支老人。

- **身份**：刘家旁支老叔伯。
- **定位**：启蒙授业者。
- **性格**：清瘦、少言、讲规矩；不热络，但做事留余地。
- **对刘枫的作用**：
  - 教他识字、修行常识、世家生存法。
  - 告诉他：在刘家，不争也要会躲。
  - 是刘枫童年少数不以出身轻慢他的人。
- **命运方向**：
  - 可在第 1-2 卷退场或老死，不必惨死。
  - 他留下的不是力量，而是刘枫的处事方式。
- **写作要点**：
  - 刘清意不要像“老爷爷”一样万能。
  - 他知道的东西有限，只是旁支老人，不是幕后大能。
  - 他越普通，越能衬出刘枫早期的真实孤弱。

### 刘岳

> 第一仇人，嫡系堂兄。

- **身份**：刘家嫡系子弟，刘枫堂兄。
- **定位**：第一卷小反派，单纯嫉妒与欺压。
- **性格**：骄横、轻慢、心胸窄；不是大恶人，但足够坏。
- **关键事件**：
  - 借“近海险地办事”之名对刘枫下狠手。
  - 直接推动刘枫濒死，被引入无名古树空间。
- **命运方向**：
  - 第一卷或第二卷被刘枫处理。
  - 死法不必华丽，越干净越好。
- **写作要点**：
  - 刘岳不需要复杂反转。
  - 他是早期压迫环境的具象，不是长期宿敌。
  - 处理刘岳，是刘枫第一次真正意识到：忍让不能换来自由。

## 道心锚：许潮

> 刘枫从小一起长大的玩伴，低阶散修后代，第 5 卷身亡。

- **身份**：青礁岛附近低阶散修后代。
- **性别**：男。
- **名字取意**：潮汐来去，象征东海童年，也象征终将失去的旧日。
- **性格**：爽朗、胆大、比刘枫外向；能把刘枫从沉默里拉出来。
- **与刘枫关系**：
  - 两人从小在海边、礁石、渔村间长大。
  - 许潮知道刘枫在刘家的处境，但不揭伤疤。
  - 刘枫不说“朋友”，但许潮是他真正的朋友。
- **修行定位**：
  - 低阶散修后代，有一点修行基础。
  - 天赋一般，但心性明亮。
  - 不与刘枫长期同行，否则容易分散主线。
- **命运**：
  - 第 5 卷因刘枫的因果牵连而死。
  - 死前不责怪刘枫。
  - 刘枫识海里有两颗道品果，却救不了他。
- **作用**：
  - 许潮之死是刘枫第一次真正的道心崩裂。
  - 从此刘枫对“自由”的理解，从“我不被安排”变成“我必须强到无人能动我所重者”。

## 海间宗人物

### 孙长老

> 不收徒，但沉默照拂刘枫。

- **身份**：海间宗外门长老或执事长老。
- **态度**：沉默照拂。
- **对刘枫的作用**：
  - 发现刘枫做事稳、心性静。
  - 不公开袒护，只在关键处给一条路。
  - 让刘枫明白：修真界并非人人都善，也并非人人都恶。
- **写作边界**：
  - 不要让他成为强力靠山。
  - 不要让他知道无名古树秘密。
  - 他只是海间宗这个小宗门里，少数愿意按规矩办事的人。

## 镜像人物

### 宋清河

> 亲宗男天骄，秩序侧镜像。

- **身份**：神州大宗出身的正道天骄。
- **气质**：端正、冷静、温润，不傲慢，不轻浮。
- **象征**：被秩序承认的自由。
- **与刘枫的关系**：
  - 不是单纯敌人。
  - 他像“如果刘枫出生在更好地方，本可以成为的人”。
  - 二人多次相遇，互相试探，互相不完全认同。
- **道心冲突**：
  - 宋清河相信规矩能庇护多数人。
  - 刘枫相信规矩常常先庇护制定规矩的人。
- **命运方向**：
  - 不必死。
  - 中后期各走各道，成为刘枫长期参照物。

### 苏凌

> 魔道女天骄，欲望侧镜像。

- **身份**：南瘴或魔道大派出身的女天骄。
- **气质**：冷、利、危险；不靠妩媚，而靠判断力和行动力压人。
- **象征**：放纵欲望后的自由。
- **与刘枫的关系**：
  - 不是道侣线。
  - 她欣赏刘枫的冷静，也嘲笑他的底线。
  - 她会不断逼问刘枫：既然求自由，为何还要被“善恶”“旧情”“故人”束缚？
- **道心冲突**：
  - 苏凌认为自由就是不受任何道德和情感约束。
  - 刘枫必须证明：有底线不是枷锁，而是“我仍是我”的边界。
- **命运方向**：
  - 不作为最终反派。
  - 她是刘枫的欲望侧镜子，最终可以各走各道。

## 终极关系：无名古树 / 原主

- 无名古树不是普通宝物，而是刘枫最大的关系对象之一。
- 它救过刘枫，养过刘枫，也可能一直在把刘枫引向原主的道路。
- 原主是“另一个曾追求自由的人”。
- 全书终局不是刘枫杀死某个外敌，而是他回答：

> **我是否会成为你？**

详见 `无名古树（金手指）.md`。

## 写作禁忌

- 不要给刘枫前期塞太多朋友。
- 不要把许潮写成工具人；第 5 卷要死，前期就必须让读者喜欢他。
- 不要把苏凌写成单纯诱惑型女性，她必须有自己的道。
- 不要让宋清河变成脸谱化正道伪君子；他应该真端正、真有道理。
- 不要让刘清意承担“老爷爷”功能。
- 不要让刘岳拖太久，他只是早期压迫的具象。
- 不要让人物关系替代主线：所有关系都服务刘枫对“自由”的理解变化。
